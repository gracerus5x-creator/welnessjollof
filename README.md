# Welness Jollof

Evidence-led supplement, nerve-health, and wellness guides — pharmacist-backed, plain language.

---

Auto-published static site powered by **Hugo + PaperMod**.
Built and deployed by [NAIRA-ENGINE](https://github.com/gracerus5x-creator).

## How it works

1. Articles are generated by the NAIRA-ENGINE pipeline (Claude + Toprank SEO)
2. Markdown files land in `content/posts/`
3. GitHub Actions builds Hugo on every push
4. GitHub Pages serves the static site

## Live URL

[https://gracerus5x-creator.github.io/welnessjollof/](https://gracerus5x-creator.github.io/welnessjollof/)

## Manual changes

Don't edit files directly in `content/posts/` — they're overwritten by the pipeline.
Theme and config tweaks (`hugo.toml`) are safe to hand-edit; they survive pipeline runs.
